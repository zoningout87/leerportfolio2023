PARALLAX FOREST

Voor de parallax bos heb ik eerst gebruikt gemaakt met alleen gebruikt gemaakt met CSS. Ik wilde eigenlijk geen javascript gebruiken, maar dat bleek later onmogelijk.

Mijn eerste parallax was met Z-index en met de Z as die dan de verschillende snelheden kan bepalen.

De parallax die met alleen Z- index wordt gemaakt, maar niet alleen gebruik van de X en Y as, maar ook met de Z- as die dan naar achteren loopt.
Alleen het probleem met dit is, is dat hij dan ook de PNG’s die ingeladen worden op verschillende hoogste zet in de Z- as en dat kun je niet gebruiken op een scroller. Dus ik moest toen wel een parallax gebruiken met Javascript en ik heb er 1 gevonden.

Zie hier tutorial

Deze heeft een parallax in vertical, maar ook 1 in horizontal. Die code heb ik gepakt, alle png’s ingeladen, en met verschillende snelheden gecodeerd.

SNEEUW EN AS

 Voor deze twee verschillende type neerslag, heb ik twee verschillende manieren gebruikt om te laten werken. Om de sneeuw te laten vallen, heb ik gebruik gemaakt van JAVASCRIPT, waarbij ik een script schreef dat de sneeuwvlokken willekeurig laat vallen op het scherm. Voor het as heb ik daarentegen gebruik gemaakt van Div containers en CSS. Op deze manier kon ik het as op een natuurlijke manier laten bewegen, met behulp van CSS-animaties. Door deze verschillende technieken te gebruiken, was ik in staat om twee verschillende soorten neerslag te creëren
