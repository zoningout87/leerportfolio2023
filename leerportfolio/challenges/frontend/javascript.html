javascript fundamentals

Om Javascript goed te begrijpen, heb ik de cursussen van Jonas Schmedtmann gevolgd via UDEMY.com. Hij heeft een effectieve en heldere manier van uitleggen, inclusief oefeningen, wat het leerproces bijzonder aangenaam maakte. Als je eenmaal de fundamentals hebt doorgelopen, biedt hij een roadmap aan waar je kan kiezen welke kant je op wilt gaan kwa programmeren. Ik heb de kant gekozen, zodat ik sneller een ander javascript library kan begrijpen.

React.js
