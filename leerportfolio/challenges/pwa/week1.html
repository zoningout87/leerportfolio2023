De eerste week van deze PWA workshop heb ik geprobeerd een PWA op te zetten, oftewel een Progressive Web App. Dit gaat gepaard met te maken van een manifest, een service worker en een beveiligde HTTPS verbinding. Maar laten we bij het eerste beginnen, de manifest.

SERVICE WORKER


Nu zijn we aangeland bij de service worker die we gaan installeren. Zoals ik het begrepen had, is het belangrijk dat deze op de "root" wordt geinstalleerd, zodat alle files kan inzien, en ook offline kan werken. Dit omdat het een normaal javascript via Dom werkt een een mainthread is, maar dit moet een andere draad worden. Dan roept javascript die in de DOM staat de service-worker op en dan kan hij worden geinstalleerd. Dit hebben we gedaan en we hebben de js.files gepakt van de workshop en deze in de juiste script.js gezet. Ik wilde eerst graag zien, wat het doet en in de toekomst kan ik zelf deze scripts schrijven.
